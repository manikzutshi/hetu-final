# Hetu App

A privacy-focused journaling companion powered by RunAnywhere SDK.

## Setup Requirements

This project relies on the RunAnywhere SDK, which must reside in the parent directory.

**Directory Structure:**
```
parent-folder/
├── hetu-final/        (this repo)
└── runanywhere-sdks/  (RunAnywhere SDK)
```

Please ensure `runanywhere-sdks` is present in the parent directory before building.
For setup instructions, refer to `runanywhere_setup_guide.md` (if available) or RunAnywhere SDK documentation.

## Features
- Local LLM & STT support via RunAnywhere
- Privacy-first design (no cloud data)
- Clean journal interface
